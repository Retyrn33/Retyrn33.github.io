GitHub+Hexo搭建个人Blog

今天写一下做这个个人blog的理由和所做的准备工作

做这个blog最直接的理由是我弟，一个极度勤奋自律的人（以后再好好谈谈我弟这个人），因为他给我看了他做的blog，主要内容是他在学编程的路上，对知识、技术的总结和分享，另外还有写自己的日常、学习方法、生活感想等等。看到他的blog的第一想法是：对喔，还能这么做。我作为一个很懒散的人，懒散了这么多年，心里再想好好努力，都有点难以坚持。这就导致了我在学习上容易时断时续，知识都是看一遍就忘，很难转换成自己的东西。而写blog做总结，这无疑是一个极好的方式，能把我学了的东西再自己梳理一遍，于是我决定向他学习，我也自己搭建一个blog，在上面写一写自己接下来要学的东西，当作是学习回顾和总结，也方便后面的复习。我想不管写的怎么样，不管梳理的怎么样，一点一滴写下，这样总能收获些。饭得一口一口吃，慢慢来，总能好好得消化掉。而且70同学讲过，你得先让自己去多说，不断地说，慢慢地就能好好地表达自己心中所想得了，我想这个道理也能用在学习上。其次就是，日常生活中，总会有不少事不少想法产生，我又是个比较负面的人，也是个奇怪的人，总会有各种想法心事，不太适合和他人诉说。那么，在这里说一下，也算是一个倾诉的途径。特别是2019年下半年这段时间，其中发生的事是我无法方下的，希望我有一天能在这里好好地诉说一下，大概吧。

那么下面来开始我的blog搭建

目标：搭建个人Blog
方式：GitHub + Hexo 框架
准备工作A:
1. GitHub
2. 注册账号
3. 新建一个repository，因为是做blog，这个名字必须是用户名.github.io
4. 下载、安装Git
5. 连接Git和GitHub
5.1. 获取SSH密钥
5.2. 登录GitHub在settings里填入密钥
准备工作B：
1. Node.js
2. 打开https://nodejs.org/en/下载并安装Node.js
3. 新建文件夹Hexo，并在里面再新建一个blog文件夹
4. 安装Hexo，右键点击GIT BASH HERE，输入命令：npm install hexo-cli -g
5. 安装完后输入hexo init
6. 初始化完成后输入：hexo install
7. npm install hexo-deployer-git --save
8. 运行hexo g -d

注1：blog搭建不成功可能是仓库连接的问题
